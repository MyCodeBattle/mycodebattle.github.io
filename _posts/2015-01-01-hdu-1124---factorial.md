## 代码


```c++
int main()
{
    LL n;
    int T;
    cin >> T;
    while (T--)
    {
        LL ans = 0;
        cin >> n;
        while (n)
        {
            ans += n / 5;
            n /= 5;
        }
        cout << ans << endl;
    }
    return 0;
}
```
