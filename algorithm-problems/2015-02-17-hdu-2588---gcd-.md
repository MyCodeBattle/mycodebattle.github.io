## 题意

找出1~n之间且gcd(x, n) >= M的个数。

## 思路

假设$X$是$N$的一个约数, 且$X >= M$, 那么设$P = \frac {N}{X}$

那么显然$gcd(X*k, N) = X, \text {1 <= k < P && gcd(k, P) = 1}$

所以$ans = \sum \phi (\frac {N}{k}), \text {k是N的约数且k>=M} $
